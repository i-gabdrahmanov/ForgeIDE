# T35 — Импортёр: безопасный повторный импорт

- **Веха:** M5
- **Зависит от:** T20, T24
- **Спека:** SD §8, SDD FR-9; ревью импортёра 2026-07-11 №4

## Скоуп

- Перед записью `ImportWriter` сравнивает каждый целевой файл с новым содержимым
  и делит на new / identical / modified; identical не перезаписываются.
- Конфликты (modified — в т.ч. промпты, правленные через инспектор T20, и
  скрипты судей) показываются списком; без явного подтверждения пользователя
  файл не трогается. Опция «пропустить конфликтующие».
- Перезаписанные по подтверждению файлы сохраняются в бэкап
  `.forgeide/import-backup/<timestamp>/` (или в задаче фиксируется явное решение
  отказаться от бэкапа в пользу git).

## Вне скоупа

- Трёхстороннее слияние содержимого (merge промптов) — только detect + confirm.

## Приёмка

- [ ] re-import без изменений в обвязке не перезаписывает ни одного файла
- [ ] re-import при локально правленном промпте показывает конфликт; без подтверждения файл не изменён
- [ ] после подтверждённой перезаписи прежняя версия доступна в бэкапе (либо решение об отказе зафиксировано в задаче)
