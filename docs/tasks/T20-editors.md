# T20 — Редакторы: промпты, судьи (trusted-путь)

- **Веха:** M4
- **Зависит от:** T10, T18
- **Спека:** SDD FR-8.1–8.3

## Скоуп

- Инспектор плитки: `TabPane` — промпт (RichTextFX, markdown-подсветка), скрипт (python),
  конфиг (форма по типу шага); diff к git-версии файла.
- Правка промпта при идущем прогоне: действует со следующего запуска шага, аудит-событие
  (кто, когда, diff-hash), пометка итерации в timeline (FR-8.2).
- Trusted-путь для судей/хуков (FR-8.3): сохранение из редактора → атомарно рабочая копия +
  harness-cache + hash-манифест (API из T18) + `harness.edited` с диффом.
- Подсветка контрактной части промпта agent-плитки (блок итогового JSON) с предупреждением
  при её правке/удалении.

## Вне скоупа

- Dry-run и предпросмотр (T21), конструктор графа (T22).
