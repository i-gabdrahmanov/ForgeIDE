# T13 — Вопросы от модели (WAITING_INPUT)

- **Веха:** M3
- **Зависит от:** T09, T12
- **Спека:** SDD FR-10, §5.2; Т-15

## Скоуп

- Обработка `pending_questions` из результата фазы: шаг → `WAITING_INPUT` (персистится,
  переживает рестарт), независимые ветки прогона продолжаются.
- Диалог на инфраструктуре T12: форма по типам (`choice` → radio, `text` → textarea),
  рендер `context`-артефакта рядом с вопросом.
- Отложенный ответ: бейдж «N вопросов» на плитке, повторное открытие.
- Перезапуск фазы: тот же промпт + канонический блок `answers`; полный текст в meta.json
  новой итерации; раунд вопросов не расходует счётчик судейских ре-итераций.
- Лимит ≤ 2 раундов на итерацию фазы; третий → эскалация: открыть промпт в редакторе /
  разбить шаг / отменить.
- Аудит: `question.asked` / `question.answered`.

## Вне скоупа

- Редактор промптов (T20) — эскалация до него показывает заглушку-ссылку.
