# T16 — Env-скоупинг + scope-diff

- **Веха:** M3
- **Зависит от:** T06, T09
- **Спека:** SDD SR-5, SR-6; Т-4, Т-11, Т-13

## Скоуп

- Env-скоупинг: секреты (git-токены, MCP-креды) в конфиге IDE (`~/.forgeide`, права 600);
  инжект в env фазы строго по `env_scope` шага; фазы без скоупа получают чистый env.
- Scope-diff: снимок до фазы (`git status --porcelain` + список untracked), после фазы —
  сверка затронутых файлов с масками `allowed_write` шага (glob); нарушение →
  `FAILED(scope)` + список файлов вне маски + инцидент в аудит.
- `head_before/head_after` в meta.json: фиксация HEAD до/после фазы, детект неожиданных
  коммитов/сбросов (Т-13).
- Кнопка «откатить лишнее» в диалоге инцидента: `git restore`/удаление untracked строго
  по списку нарушений, с подтверждением.

## Вне скоупа

- Outward-экшены (T17), OS-sandbox (пост-MVP).
