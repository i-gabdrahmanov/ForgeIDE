# T12 — Гейты человека

- **Веха:** M3
- **Зависит от:** T06, T10
- **Спека:** SDD FR-5; Т-5, Т-16

## Скоуп

- Исполнение `GateStep`: шаг → `WAITING_GATE`, модальный диалог: вопрос, варианты,
  рендер артефактов (markdown) и `git diff` — **реальные данные с диска**, не пересказ модели.
- Текст модели (summary) в диалоге визуально маркирован как untrusted (FR-5.2).
- Чекбокс «я посмотрел дифф» для шагов с риском R2+ по профилю критичности (FR-5.3).
- Ответ → `GateAnswered(user, ts)` → SoT + аудит; отклонение → ветка `routes` или возврат.
- Отложенный ответ: диалог можно закрыть, шаг остаётся WAITING_GATE, повторное открытие с канваса.
- Диалог эскалации FR-11.3 (общая инфраструктура с гейтами): история errors.json по итерациям,
  diff артефактов, действия повторить/правка промпта/сброс цепочки/отмена/override (причина обязательна).

## Вне скоупа

- Вопросы модели (T13) — но диалоговая инфраструктура делается здесь переиспользуемой.
