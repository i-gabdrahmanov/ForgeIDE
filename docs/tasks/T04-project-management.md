# T04 — Проекты и рантаймы

- **Веха:** M1
- **Зависит от:** T01, T02
- **Спека:** SDD FR-1; БТ §4.1

## Скоуп

- Модель проекта + реестр `~/.forgeide/projects.json`: имя, путь репозитория, спецификации,
  params, профиль критичности (low/medium/high → R2/R1/R0).
- Форма создания/редактирования проекта; динамические поля из `pipeline.yaml → params`
  (required/hint), расширяемые произвольными key-value.
- Менеджер рантаймов: путь к бинарю, флаги (`--experimental-hooks`), проверка доступности
  (`--version`), статус в UI.
- Экран списка проектов, открытие проекта.

## Вне скоупа

- Деплой обвязки и preflight (T18), запуск прогонов (T06+).

## Приёмка

- [ ] созданный проект персистится, после рестарта IDE открывается со всеми полями
- [ ] несуществующий бинарь рантайма → понятная ошибка в UI, проект сохранить можно, запуск помечен недоступным
- [ ] params с `required: true` без значения блокируют запуск прогона
